# TMVE Tracker

App Android para rastreo de estaciones Telefónica Venezuela en tiempo real.

## Cómo compilar

Este repositorio compila automáticamente la APK con GitHub Actions.

1. Haz push a la rama `main`
2. Ve a la pestaña **Actions** en GitHub
3. Espera ~15 minutos a que termine
4. Descarga la APK desde **Artifacts → TMVETracker-APK**

## Características
- 3,189 estaciones TMVE cargadas internamente
- GPS en tiempo real
- Voz anuncia el nombre de la estación al cambiar
- Historial de ruta exportable
